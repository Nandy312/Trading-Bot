# Trading Bot Dashboard (React)

## Features
- Real-time bot status
- Asset selection and trade controls
- Trade history
- Live price/tick chart

## Getting Started
1. Install Node.js and npm
2. Run `npm install` in this folder
3. Run `npm start` to launch the dashboard

## Pages
- Dashboard: Bot status, controls
- Trade: Asset selection, trade form
- History: Trade history
- Chart: Live price/tick chart

## API Endpoints (served by FastAPI backend)
- `/status` - Bot status
- `/trade` - Place trade
- `/history` - Trade history
- `/start` - Start bot
- `/stop` - Stop bot
